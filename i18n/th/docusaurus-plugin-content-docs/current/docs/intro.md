# ยินดีต้อนรับสู่ศูนย์ช่วยเหลือ Forex Margin

ศูนย์ช่วยเหลือนี้ให้ความรู้ที่ครอบคลุมเกี่ยวกับการเทรด Forex Margin ช่วยให้คุณเข้าใจฟีเจอร์ของแพลตฟอร์ม กฎการเทรด และวิธีการจัดการความเสี่ยงได้อย่างรวดเร็ว

## นำทางด่วน

| ประเภท | คำอธิบาย |
|------|------|
| 🚀 **[คู่มือเริ่มต้น](/docs/getting-started/intro)** | เริ่มจากศูนย์ เสร็จสมบูรณ์การลงทะเบียนบัญชี การยืนยัน และการฝากเงินครั้งแรก |
| 📊 **[พื้นฐานการเทรด](/docs/trading-basics/intro)** | เข้าใจแนวคิดหลัก เช่น มาร์จิ้น เลเวอเรจ สเปรด ประเภทคำสั่ง |
| 💰 **[การฝากและถอนเงิน](/docs/deposits-withdrawals/intro)** | วิธีการชำระเงิน ค่าธรรมเนียม เวลาดำเนินการ และขีดจำกัด |
| 🛡️ **[การจัดการความเสี่ยง](/docs/risk-management/intro)** | การเรียกมาร์จิ้น กลไกการล้างบัญชี และกลยุทธ์ควบคุมความเสี่ยง |
| 🔧 **[การจัดการบัญชี](/docs/account-management/intro)** | ประเภทบัญชี การตั้งค่าความปลอดภัย และการจัดการขีดจำกัดการเทรด |
| ❓ **[คำถามที่พบบ่อย](/docs/faq/account)** | ปัญหาที่ผู้ใช้พบเจอบ่อยที่สุดและวิธีแก้ไข |

## ต้องการความช่วยเหลือจากคน?

หากคุณไม่พบคำตอบที่ต้องการในเอกสาร คุณสามารถติดต่อทีมสนับสนุนของเราผ่านปุ่ม **💬 แชทสนับสนุนออนไลน์** ที่มุมขวาล่าง เราจะให้ความช่วยเหลือคุณโดยเร็วที่สุด
