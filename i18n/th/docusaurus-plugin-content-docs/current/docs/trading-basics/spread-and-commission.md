# 点差与佣金

了解交易成本是制定交易策略的重要前提。

## 什么是点差？

点差（Spread）是买入价（Ask）和卖出价（Bid）之间的差额，是外汇交易最主要的成本。

```
点差 = Ask Price - Bid Price
```

### 点差示例

EUR/USD 报价：Bid 1.0848 / Ask 1.0850

- **买入价 (Ask)**：1.0850 — 您做多时以此价格买入
- **卖出价 (Bid)**：1.0848 — 您做空时以此价格卖出
- **点差**：2 pips (1.0850 - 1.0848 = 0.0002)

:::info 什么是 Pip？
Pip（百分比点）是外汇价格变动的最小单位。对于大多数货币对，1 pip = 0.0001（小数点后第4位）。对于日元相关货币对，1 pip = 0.01。
:::

## 点差类型

| 类型 | 说明 | 适合人群 |
|------|------|----------|
| **固定点差** | 点差始终保持不变 | 风险偏好低的交易者 |
| **浮动点差** | 点差随市场波动变化 | 追求更低交易成本的交易者 |

## 佣金

ECN 账户不收取点差，但收取每笔交易的佣金：

- **外汇**：$3.5 / 标准手 / 单边（开仓+平仓 = $7/手）
- **贵金属**：$5 / 标准手 / 单边
- **零佣金**：Standard 和 Pro 账户免佣金

## 隔夜费（掉期）

持仓过夜会产生隔夜费（Swap）：

- **正掉期**：持仓过夜可获得费用
- **负掉期**：持仓过夜需要支付费用

隔夜费在每天服务器时间 00:00 自动扣除或添加。
