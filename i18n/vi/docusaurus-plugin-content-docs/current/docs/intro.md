# Chào mừng đến với Trung tâm Trợ giúp Forex Ký quỹ

Trung tâm trợ giúp này cung cấp kho kiến thức toàn diện về giao dịch Forex ký quỹ, giúp bạn nhanh chóng hiểu về tính năng nền tảng, quy tắc giao dịch và phương pháp quản lý rủi ro.

## Điều hướng nhanh

| Phân loại | Mô tả |
|------|------|
| 🚀 **[Hướng dẫn bắt đầu](/docs/getting-started/intro)** | Từ số 0, hoàn thành đăng ký tài khoản, xác minh và nạp tiền lần đầu |
| 📊 **[Cơ bản giao dịch](/docs/trading-basics/intro)** | Hiểu các khái niệm cốt lõi như ký quỹ, đòn bẩy, chênh lệch, loại lệnh |
| 💰 **[Nạp và rút tiền](/docs/deposits-withdrawals/intro)** | Phương thức thanh toán, phí, thời gian xử lý và giới hạn |
| 🛡️ **[Quản lý rủi ro](/docs/risk-management/intro)** | Bổ sung ký quỹ, cơ chế thanh lý và chiến lược kiểm soát rủi ro |
| 🔧 **[Quản lý tài khoản](/docs/account-management/intro)** | Loại tài khoản, cài đặt bảo mật và quản lý giới hạn giao dịch |
| ❓ **[Câu hỏi thường gặp](/docs/faq/account)** | Các vấn đề người dùng thường gặp nhất và giải pháp |

## Cần hỗ trợ trực tiếp?

Nếu bạn không tìm thấy câu trả lời trong tài liệu, bạn có thể liên hệ với đội ngũ hỗ trợ của chúng tôi thông qua nút **💬 Hỗ trợ trực tuyến** ở góc dưới bên phải, chúng tôi sẽ hỗ trợ bạn sớm nhất có thể.
