# Forex証拠金ヘルプセンターへようこそ

このヘルプセンターは、包括的なForex証拠金取引のナレッジベースを提供し、プラットフォーム機能、取引ルール、リスク管理方法をすばやく理解できるようにサポートします。

## クイックナビゲーション

| カテゴリー | 説明 |
|------|------|
| 🚀 **[はじめに](/docs/getting-started/intro)** | ゼロから始めて、アカウント登録、本人確認、初回入金を完了 |
| 📊 **[取引の基礎](/docs/trading-basics/intro)** | 証拠金、レバレッジ、スプレッド、注文タイプなどの基本概念を理解 |
| 💰 **[入金と出金](/docs/deposits-withdrawals/intro)** | 支払い方法、手数料、処理時間、限度額の説明 |
| 🛡️ **[リスク管理](/docs/risk-management/intro)** | マージンコール、強制決済メカニズム、リスクコントロール戦略 |
| 🔧 **[アカウント管理](/docs/account-management/intro)** | アカウントタイプ、セキュリティ設定、取引限度額管理 |
| ❓ **[よくある質問](/docs/faq/account)** | ユーザーが最もよく遭遇する問題と解決策 |
