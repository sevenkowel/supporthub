# MT5 "Invalid Price" 无效价格错误

在 MT5 下单时收到 **"Invalid Price"（无效价格）** 错误是一个常见问题。本文将告诉您为什么会出现这个错误，以及如何快速解决。

## ⚡ 快速解决方法

> 如果您正在交易中，请先尝试以下快速修复：
> 1. **刷新报价** — 右键图表 → 刷新
> 2. **改用市价单** — 删除当前挂单，改为市价开仓
> 3. **重新连接平台** — 断开后重新连接服务器

---

## 🔍 Invalid Price 的常见原因

### 原因 1：市场价格已偏离您的挂单价格

下挂单时，如果市场价格已经大幅移动，您设定的价格可能已经不在有效范围内。

**示例**：
- 您设置 Buy Limit @ 1.2000
- 当前市价已涨至 1.2150
- 您的挂单价低于当前市价的合理范围 → 报 Invalid Price

**解决**：调整挂单价格至当前市价附近的合理位置。

---

### 原因 2：止损/止盈距离过近（违反最小距离规则）

MT5 对止损和止盈有**最小距离（Stop Level）**限制，通常为 0–20 个点不等。

**检查方法**：
1. 右键品种 → 品种规格
2. 查看"止损位"（Stop Level）字段

**解决**：将止损/止盈设置在当前价格 + Stop Level 之外。

---

### 原因 3：价格精度（小数位）不正确

MT5 对价格精度有严格要求，通常为 5 位小数（部分品种 3 位）。

**错误示例**：输入 `1.2` 而非 `1.20000`  
**解决**：确保价格输入正确的小数位数，或让平台自动填充。

---

### 原因 4：市场已关闭

**周末** 或 **节假日** 尝试下单会收到 Invalid Price，因为市场未开盘，没有有效报价。

**解决**：查看[交易时间](/docs/trading-basics/trading-hours)，在市场开盘后下单。

---

### 原因 5：挂单价格与当前价格方向相反

MT5 对不同类型的挂单有价格方向要求：

| 挂单类型 | 正确价格位置 |
|----------|------------|
| **Buy Limit** | 必须低于当前市价 |
| **Buy Stop** | 必须高于当前市价 |
| **Sell Limit** | 必须高于当前市价 |
| **Sell Stop** | 必须低于当前市价 |

如果方向设置错误，MT5 会报 Invalid Price。

---

### 原因 6：异常波动期间的报价问题

重要数据发布（如非农、CPI）期间，市场波动剧烈，报价快速跳动，可能导致您提交的价格瞬间失效。

**解决**：数据公布后等待市场稳定，再重新下单。

## 🛠️ 分步排查指南

```
收到 Invalid Price 错误？

Step 1: 检查当前市价
  → 确认价格是否大幅偏移？
  → Yes → 更新挂单价格
  → No → 继续下一步

Step 2: 检查止损/止盈距离
  → 是否符合最小距离要求？
  → No → 调整 SL/TP 位置
  → Yes → 继续下一步

Step 3: 检查市场状态
  → 市场是否正在开盘？
  → No（周末/节假日）→ 等待开盘
  → Yes → 继续下一步

Step 4: 检查挂单类型与价格方向
  → Buy Limit 是否低于市价？
  → Sell Limit 是否高于市价？
  → No → 修正方向

Step 5: 重启平台重试
  → 断开连接，重新登录
  → 刷新报价后重新下单
```

## ⚙️ MT5 平台设置优化

### 开启"偏差"（Deviation）容忍度

允许订单在一定范围内浮动执行，减少因价格变动导致的失败：

1. 打开 MT5 下单窗口
2. 勾选 **"允许偏差"**
3. 设置偏差点数（建议 5–20 点）

### 使用市价单替代挂单

当市场波动较大时，改用市价单可以避免 Invalid Price：

- 市价单（Market Order）→ 按当前最优价格立即执行
- 适合快速进场，不在意几个点的滑点

## ❓ 常见问题

### 为什么之前能下单，现在突然报 Invalid Price？
市场价格发生了变化。如果是挂单，检查挂单价格是否仍然合理；如果是止损/止盈，检查是否符合最小距离要求。

### 我设置的价格完全正确，为什么还是报错？
可能是平台连接问题导致报价不同步。请断开重连，刷新报价后重试。

### MT5 和 MT4 的 Invalid Price 处理方式有区别吗？
基本原因相同，但 MT5 新增了 Buy Stop Limit 和 Sell Stop Limit 两种挂单类型，这两种类型有额外的价格规则需要注意。
