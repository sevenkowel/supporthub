# 保证金追缴与强平

当账户权益下降到特定水平时，平台将启动风险保护机制。

## 保证金比例

```
保证金比例 = (账户净值 ÷ 已用保证金) × 100%
```

## 两个关键阈值

### 保证金追缴 (Margin Call) — 100%

当保证金比例降至 **100%** 或以下时：

- 系统发送 Margin Call 通知（邮件 + 短信）
- 平台建议您追加保证金或减仓
- 您仍然可以开新仓（如果保证金允许）
- **不会自动平仓**

:::caution 注意
收到 Margin Call 说明您的账户已处于高风险状态。应立即采取措施：追加资金或减少持仓。
:::

### 强行平仓 (Stop Out) — 30%

当保证金比例降至 **30%** 或以下时：

- 系统将自动强制平仓
- 从亏损最大的仓位开始逐个平仓
- 直到保证金比例恢复到 30% 以上
- **此过程不可撤销**

## 实例说明

| 项目 | 数值 |
|------|------|
| 余额 | $10,000 |
| 杠杆 | 1:100 |
| 持仓 | 1 标准手 EUR/USD |
| 已用保证金 | $1,085 |

**场景：EUR/USD 下跌 85 pips（浮亏 $850）**

```
净值 = $10,000 - $850 = $9,150
保证金比例 = $9,150 ÷ $1,085 = 843% → 安全 ✅
```

**场景：EUR/USD 下跌 891 pips（浮亏 $8,915）**

```
净值 = $10,000 - $8,915 = $1,085
保证金比例 = $1,085 ÷ $1,085 = 100% → Margin Call ⚠️
```

**场景：EUR/USD 下跌 924 pips（浮亏 $9,240）**

```
净值 = $10,000 - $9,240 = $760
保证金比例 = $760 ÷ $1,085 = 70% → 接近 Stop Out 🚨
```

## 如何避免被强平？

1. 严格设置止损
2. 控制仓位大小（建议已用保证金不超过余额的 20%）
3. 保持充足的可用保证金
4. 关注保证金比例变化
5. 设置保证金比例预警
