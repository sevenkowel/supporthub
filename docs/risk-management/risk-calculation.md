# 风险计算

学会计算交易风险是控制资金的关键。

## 仓位大小计算

### 公式

```
仓位大小 = (账户余额 × 风险百分比) ÷ (止损点数 × 每点价值)
```

### 示例

- 账户余额：$10,000
- 风险比例：1%（即 $100）
- 止损距离：50 pips
- 交易 EUR/USD（每点价值 = $10/标准手）

```
仓位大小 = $100 ÷ (50 × $10) = 0.2 标准手
```

## 每点价值参考

| 货币对 | 1 标准手 | 0.1 迷你手 | 0.01 微型手 |
|--------|----------|-----------|-----------|
| EUR/USD | $10.00 | $1.00 | $0.10 |
| GBP/USD | $10.00 | $1.00 | $0.10 |
| USD/JPY | $6.67 | $0.67 | $0.07 |
| USD/CHF | $10.00 | $1.00 | $0.10 |
| AUD/USD | $10.00 | $1.00 | $0.10 |

:::info 注意
每点价值会随汇率变化而波动。以上为近似值。
:::

## 风险回报比

```
风险回报比 = 止损距离 ÷ 止盈距离
```

| 风险回报比 | 评价 |
|-----------|------|
| 1:1 | 最低可接受 |
| 1:1.5 | 一般 |
| 1:2 | 良好 |
| 1:3+ | 优秀 |

:::tip 建议
每笔交易的预期风险回报比至少为 1:2，即潜在盈利至少是潜在亏损的 2 倍。
:::

## 综合风险管理规则

| 规则 | 说明 |
|------|------|
| 单笔风险 | 不超过余额的 1-2% |
| 总风险敞口 | 不超过余额的 5-10% |
| 最大同时持仓 | 不超过 3-5 个 |
| 日内最大亏损 | 不超过余额的 3-5% |
| 连续亏损 | 连续 3 次亏损后暂停交易，复盘分析 |
