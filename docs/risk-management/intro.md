# 风险管理

外汇保证金交易具有较高的杠杆风险。有效的风险管理是长期交易成功的关键。

:::danger 重要风险提示
外汇保证金交易可能导致损失全部初始保证金，甚至超过初始保证金。请在充分了解风险、仅使用可承受损失的资金进行交易。
:::

## 风险管理核心原则

1. **永远设置止损** — 每笔交易都必须有明确的止损点
2. **控制仓位大小** — 单笔交易风险不超过总资金的 1-2%
3. **避免过度杠杆** — 根据经验选择合适的杠杆比例
4. **分散风险** — 不要将所有资金集中在单一交易
5. **保持理性** — 遵循交易计划，避免情绪化交易
