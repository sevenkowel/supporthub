# 入金方式详解

## 银行电汇

适用于大额入金，无金额上限。

**汇款信息（入金后系统自动生成）：**

| 项目 | 说明 |
|------|------|
| 受益人名称 | 平台托管银行账户 |
| 银行名称 | 国际一级银行 |
| SWIFT 代码 | 在入金页面获取 |
| 账号 | 在入金页面获取 |
| 参考编号 | 您的交易账号 |

:::info 汇款备注
请在汇款时务必填写交易账号作为备注，以便快速匹配入金。
:::

## 信用卡 / 借记卡

支持 Visa 和 Mastercard。

- ✅ 即时到账
- ⚠️ 3D Secure 验证可能需要
- ⚠️ 每日累计限额 $20,000
- ⚠️ 部分银行可能限制外汇交易

## 加密货币 (USDT TRC20)

使用 Tether (USDT) 通过 TRC20 网络入金。

1. 在入金页面选择 USDT
2. 获取专属收款地址
3. 从您的钱包转账 USDT
4. 等待网络确认（6-20 个区块）

:::caution 注意
- 仅支持 TRC20 网络
- 请勿向同一地址发送其他代币
- 最低入金：50 USDT
- 网络矿工费：约 1 USDT
:::

## 电子钱包

### Skrill

1. 登录 Skrill 账户
2. 选择向商户付款
3. 输入入金金额和邮箱
4. 确认付款

### Neteller

流程与 Skrill 类似，支持即时到账。
