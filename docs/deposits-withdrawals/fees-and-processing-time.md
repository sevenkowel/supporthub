# 手续费与处理时间

## 费用概览

| 项目 | Standard | Pro | ECN |
|------|----------|-----|-----|
| 入金手续费 | 按支付方式 | 按支付方式 | 按支付方式 |
| 出金手续费 | 按支付方式 | 按支付方式 | 按支付方式 |
| 交易佣金 | 无 | 无 | $3.5/手/单边 |
| 隔夜费 | 有 | 有 | 有 |
| 账户管理费 | 无 | 无 | 无 |
| 闲置费 | 无 | 无 | 无 |

## 入金到账时间

| 支付方式 | 工作日 | 周末/假日 |
|----------|--------|----------|
| 银行电汇 | 1-3 工作日 | 下一个工作日开始计算 |
| 信用卡 | 即时 | 即时 |
| USDT | 10-30 分钟 | 10-30 分钟 |
| 电子钱包 | 即时 | 即时 |
| 本地银行 | 1 工作日 | 下一个工作日 |

## 出金处理时间

| 支付方式 | 正常情况 | 峰值期间 |
|----------|----------|----------|
| 银行电汇 | 1-3 工作日 | 3-5 工作日 |
| 信用卡退款 | 3-5 工作日 | 5-7 工作日 |
| USDT | 1-2 小时 | 2-4 小时 |
| 电子钱包 | 即时 | 1 小时 |

## 免费出金额度

Standard 和 Pro 账户客户每月享有 **1 次免费银行电汇出金**，超出部分收取 $25/笔。

ECN 账户客户享有 **无限次免费出金**。

:::info 如何查看费用
登录客户中心，在 **"交易历史"** 页面可以查看所有交易和资金往来的详细费用明细。
:::
