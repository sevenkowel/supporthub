# 出金方式详解

## 出金申请步骤

1. 登录 **客户中心**
2. 选择 **"出金"**
3. 选择出金方式
4. 输入出金金额
5. 确认并提交申请

## 出金审核

所有出金申请需要经过安全审核：

| 审核阶段 | 时间 | 说明 |
|----------|------|------|
| 系统检查 | 即时 | 检查账户状态和出金金额 |
| 安全审核 | 1-2 小时 | 反洗钱和欺诈检查 |
| 财务处理 | 视支付方式而定 | 实际转账处理 |

## 各方式出金详情

### 银行电汇

- **手续费**：$25 / 笔
- **处理时间**：1-3 工作日
- **最低金额**：$100
- **适用场景**：大额出金、利润提取

### 信用卡退款

- **手续费**：免费
- **处理时间**：3-5 工作日
- **最高金额**：不超过原始入金金额
- **适用场景**：退还信用卡入金部分

### 加密货币 (USDT TRC20)

- **手续费**：1 USDT
- **处理时间**：1-2 小时
- **最低金额**：50 USDT

:::warning 出金注意事项
- 确保出金地址准确无误
- 请勿使用交易所地址（部分交易所不支持合约交易退款）
- 大额出金（超过 $10,000）可能需要额外验证
:::
