# 订单执行常见问题

**Q1: 下单后多久成交？**  
A: 市价单通常在毫秒内成交。挂单需等待市场价格达到设定值。

**Q2: 止损单一定按设定价格执行吗？**  
A: 在正常市场条件下，止损单将按设定价格执行。但在市场出现跳空或流动性不足时，可能以较差价格成交。

**Q3: 可以修改已挂的单吗？**  
A: 可以。在交易平台上右键点击挂单，选择"修改"即可调整价格、止损和止盈。

**Q4: 挂单会自动过期吗？**  
A: 默认挂单为 GTC（Good Till Cancel），不会自动过期。您也可以设置为当日有效或指定时间过期。

**Q5: 为什么我的止盈没有触发？**  
A: 请检查：
- 止盈设置是否正确
- 订单是否仍然有效（未被手动平仓）
- 市场价格是否实际到达了止盈价格
