# 滑点常见问题

**Q1: 什么是滑点？**  
A: 滑点是订单请求价格与实际成交价格之间的差异。当市场快速变动或流动性不足时，可能出现正向滑点（更有利）或负向滑点（更不利）。

**Q2: 为什么会出现滑点？**  
A: 主要原因包括：
- 重大经济数据发布（如非农、利率决议）
- 市场流动性不足
- 网络延迟
- 周末开盘跳空

**Q3: ECN 账户的滑点更小吗？**  
A: ECN 账户连接银行间市场，通常能获得更好的执行价格。但极端市场条件下仍可能出现滑点。

**Q4: 如何减少滑点？**  
A: 建议：
- 避免在重大数据发布时交易
- 使用限价单代替市价单
- 选择流动性高的时段交易
- 确保网络连接稳定
