# 交易相关常见问题

**Q1: 最小交易手数是多少？**  
A: 所有账户类型的最低手数均为 0.01 Lot（微型手）。

**Q2: 什么是杠杆？我可以修改吗？**  
A: 杠杆是放大您交易能力的工具。Standard 账户最高 1:100，Pro 最高 1:200，ECN 最高 1:500。您可以在客户中心申请调整杠杆。

**Q3: 为什么我的订单被拒绝？**  
A: 可能原因：
- 保证金不足
- 市场休市时间
- 价格距离止损/止盈太近
- 账户处于受限状态

**Q4: 什么是滑点？**  
A: 滑点是订单执行价格与预期价格的差异。在剧烈市场波动或流动性低时可能出现。

**Q5: 支持剥头皮交易吗？**  
A: 支持。所有账户类型都允许剥头皮交易，ECN 账户尤其适合。

**Q6: 支持 EA 自动交易吗？**  
A: 支持。MT4 和 MT5 都支持 EA（Expert Advisor）自动交易。
