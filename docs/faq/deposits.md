# 入金常见问题

**Q1: 入金需要多长时间到账？**  
A: 信用卡、电子钱包和加密货币通常即时到账。银行电汇需要 1-3 个工作日。

**Q2: 为什么入金失败了？**  
A: 常见原因：
- 信用卡发卡行拒绝（可能需要先联系银行开通外汇交易权限）
- 支付金额超过限额
- 银行卡信息填写有误
- 加密货币转入错误的网络

**Q3: 入金有手续费吗？**  
A: 平台对银行电汇和 USDT 入金不收取手续费（可能产生网络矿工费）。信用卡和电子钱包由支付服务商收取少量费用。

**Q4: 可以用他人银行卡入金吗？**  
A: 不可以。所有入金必须使用账户持有人本人的支付方式。违反此规定可能导致账户被冻结。

**Q5: 入金金额有限制吗？**  
A: 不同支付方式有不同的最低和最高金额限制，请参考 [入金方式详解](/docs/deposits-withdrawals/deposit-methods)。
