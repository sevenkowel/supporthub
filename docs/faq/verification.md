# 身份验证常见问题

**Q1: 哪些文件可以作为身份证明？**  
A: 有效的护照、身份证或驾驶执照。文件需在有效期内，照片清晰。

**Q2: 哪些文件可以作为地址证明？**  
A: 近 3 个月内的水电费账单、银行对账单或政府信件。文件上需显示您的姓名和当前居住地址。

**Q3: 验证需要多长时间？**  
A: 工作日提交通常 1-2 小时内完成。周末和节假日提交将在下一个工作日处理。

**Q4: 验证失败怎么办？**  
A: 查看验证页面的失败原因，修正后重新提交即可。常见失败原因包括文件不清晰、已过期或姓名不一致。

**Q5: 可以修改已验证的信息吗？**  
A: 已验证的姓名和出生日期不可修改。如需修改，请联系 [在线客服](javascript:void(0)) 提供相关证明文件。
