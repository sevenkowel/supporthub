# 登录问题常见问题

**Q1: 无法登录客户中心怎么办？**  
A: 请尝试以下步骤：
1. 确认使用正确的邮箱地址
2. 检查密码大小写是否正确
3. 清除浏览器缓存后重试
4. 尝试使用其他浏览器
5. 检查网络连接

**Q2: 登录 MT5 时提示"无效账号"？**  
A: 请确认：
- 登录 ID 正确（不是邮箱）
- 选择了正确的服务器
- 密码区分大小写

**Q3: 收到"异地登录"警告怎么办？**  
A: 如果是您本人操作，可以直接忽略。如非本人操作，请立即修改密码并联系 [在线客服](javascript:void(0))。

**Q4: 启用 2FA 后无法登录？**  
A: 如果无法使用 Google Authenticator：
1. 检查手机时间是否准确
2. 使用备份码登录
3. 联系客服验证身份后重置 2FA
