# 下载交易平台

我们提供多种交易平台选择，满足不同交易者的需求。

## 可用平台

### MetaTrader 5 (推荐)

| 特性 | 说明 |
|------|------|
| 支持系统 | Windows / macOS / Linux / iOS / Android |
| 核心功能 | 高级图表、EA 自动交易、深度市场数据 |
| 交易品种 | 外汇、贵金属、原油、股指、加密货币 |
| 适合人群 | 所有级别交易者 |

:::info 推荐
MT5 是最全面的选择，支持所有交易品种和高级功能。如果您不确定选择哪个，建议从 MT5 开始。
:::

### MetaTrader 4

| 特性 | 说明 |
|------|------|
| 支持系统 | Windows / macOS / iOS / Android |
| 核心功能 | 经典图表、EA 交易、自定义指标 |
| 交易品种 | 外汇、贵金属 |
| 适合人群 | 习惯 MT4 界面的交易者 |

### WebTrader（网页版）

无需下载安装，通过浏览器直接交易：

- 支持 Chrome、Firefox、Safari、Edge
- 基础图表和技术分析工具
- 所有主要交易品种
- 适合临时使用或无法安装软件的场景

## 登录交易账户

1. 安装并打开 MT5
2. 在登录窗口中：
   - **服务器**：选择我们的服务器
   - **登录 ID**：您的交易账户编号（如 12345678）
   - **密码**：您设置的交易密码
3. 点击 **"登录"**

:::tip 首次登录
首次登录后建议立即：
- 修改默认密码
- 设置交易通知
- 熟悉图表工具和下单功能
:::
