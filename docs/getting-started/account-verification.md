# 账户验证（KYC）

根据国际反洗钱（AML）法规，所有客户必须完成身份验证（Know Your Customer）后才能入金和交易。

## 所需材料

### 个人账户

| 材料 | 要求 | 格式 |
|------|------|------|
| 身份证明 | 有效的身份证 / 护照 / 驾照 | JPG/PNG，小于 5MB |
| 地址证明 | 近 3 个月内的水电费账单 / 银行对账单 | JPG/PNG/PDF，小于 5MB |

### 企业账户

除上述材料外，还需提供：

- 企业注册证明
- 公司章程
- 董事会成员身份证明
- 企业地址证明

## 提交验证

1. 登录您的账户
2. 进入 **"账户设置" → "身份验证"**
3. 按提示上传所需文件
4. 提交审核

## 审核时间

| 提交时间 | 预计审核时间 |
|----------|-------------|
| 工作日 09:00-18:00 | 1-2 小时 |
| 工作日其他时间 | 4-6 小时 |
| 周末及节假日 | 下一个工作日 |

:::tip 加快审核速度
请确保上传的文件清晰完整：
- 所有文字清晰可读
- 文件四角完整可见
- 无反光、遮挡或修改痕迹
- 文件在有效期内
:::

## 验证失败常见原因

1. **文件不清晰** — 请重新拍摄或扫描
2. **文件已过期** — 请使用有效期内的证件
3. **地址证明日期超期** — 请提供近 3 个月内的账单
4. **姓名不一致** — 确保证件姓名与注册姓名完全一致

验证失败后，您可以在验证页面查看具体原因并重新提交。
