# 入门指南

欢迎开始您的外汇保证金交易之旅！本指南将带您完成从账户注册到第一笔交易的完整流程。

## 概览

外汇保证金交易是一种利用杠杆放大交易规模的金融衍生品交易方式。通过缴纳一定比例的保证金，您可以交易价值远大于保证金的货币对。

:::tip 开始之前
请确保您已年满 18 周岁，并了解外汇交易的基本风险。外汇保证金交易可能带来重大损失。
:::

## 快速开始步骤

1. **[注册账户](/docs/getting-started/account-registration)** — 填写基本信息完成注册
2. **[身份验证](/docs/getting-started/account-verification)** — 上传身份证明文件完成 KYC 认证
3. **[首次入金](/docs/getting-started/first-deposit)** — 选择支付方式完成首次资金入账
4. **[下载平台](/docs/getting-started/platform-download)** — 安装交易客户端开始交易

## 平台核心数据

| 项目 | 说明 |
|------|------|
| 可交易品种 | 50+ 外汇货币对、黄金、白银、原油、股指 |
| 最大杠杆 | 1:500 |
| 最低手数 | 0.01 Lot（微型手） |
| 交易时间 | 周一 00:00 — 周六 05:00（服务器时间） |
| 执行方式 | 市价执行（Market Execution） |

:::warning 风险提示
外汇保证金交易具有较高的杠杆风险。您可能损失全部初始保证金，甚至超过初始保证金。请在充分了解风险的情况下进行交易。
:::
