# MT5平台全面指南

- **外汇**: 100+货币对
- **股票**: 全球主要交易所股票
- **期货**: 商品、指数期货合约
- **加密货币**: 主流数字资产
- **期权**: 金融衍生品交易

## 交易功能详解

### 🆕 MT5特有订单类型

1. **Buy Stop Limit（买入止损限价单）**
   - 触发价格到达后，以限价单执行
   - 适用于突破交易策略

2. **Sell Stop Limit（卖出止损限价单）**
   - 触发价格到达后，以限价单执行
   - 保护性止盈策略

### 订单执行模式
- **立即执行**: 当前市场价立即成交
- **请求执行**: 向服务器请求报价后执行
- **市场执行**: 最接近当前价的可用价格

### 持仓管理
- **对冲模式**: 同一品种可有多方向持仓
- **净额模式**: 自动计算净持仓
- **部分平仓**: 可平仓部分持仓数量
- **批量操作**: 同时管理多个订单

### 常见问题解决

| 问题 | MT5特有原因 | 解决方案 |
|------|-------------|----------|
| **部分成交处理** | MT5支持部分成交 | 调整订单量或使用全额执行 |
| **对冲模式切换** | 账户模式设置 | 工具 → 选项 → 交易 → 账户模式 |
| **深度行情延迟** | 数据源问题 | 检查数据订阅，切换服务器 |
| **MQL5兼容性** | EA需要重新编译 | 使用MetaEditor 5重新编译 |
| **多市场连接** | 不同市场数据源 | 分别订阅各市场数据 |
