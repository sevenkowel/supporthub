# MT4平台使用指南

MetaTrader 4（MT4）是全球最受欢迎的外汇交易平台，以其稳定性和丰富的功能著称。本指南将帮助您全面掌握MT4的使用。

## 快速入门

### 首次登录
1. **打开MT4平台** - 从开始菜单或桌面快捷方式启动
2. **添加交易账户** - 点击左上角"文件" → "登录到交易账户"
3. **输入账户信息** - 填写服务器、账号、密码
4. **选择服务器** - 从列表中选择合适的交易服务器
5. **连接成功** - 右下角显示绿色笑脸图标

### 界面布局介绍


## 核心功能详解

### 📈 图表操作
- **添加图表**: 市场报价窗口右键 → "图表窗口"
- **时间周期**: 工具栏选择M1/M5/M15/H1/H4/D1/W1/MN
- **图表类型**: 蜡烛图、柱状图、折线图
- **缩放平移**: 鼠标滚轮缩放，按住右键拖动平移

### 📊 技术分析工具
- **添加指标**: 插入 → 技术指标 → 选择指标
- **常用指标**:
  - 移动平均线（MA）
  - 布林带（Bollinger Bands）
  - 相对强弱指数（RSI）
  - 随机指标（Stochastic）
  - MACD指标

- **图形对象**:
  - 趋势线、支撑阻力线
  - 斐波那契回调线
  - 文本标注、图形标注

### 💰 交易执行
- **市场执行**: 右键图表 → 交易 → 新订单
- **挂单交易**: 4种挂单类型：
  1. Buy Limit（买入限价）
  2. Sell Limit（卖出限价）
  3. Buy Stop（买入止损）
  4. Sell Stop（卖出止损）

- **订单参数**:
  - 交易量（手数）
  - 止损（Stop Loss）
  - 止盈（Take Profit）
  - 注释（可选）

### 🔔 警报和通知
- **价格警报**: 右键价格 → "设置警报"
- **指标警报**: 某些指标支持警报功能
- **弹出通知**: 平台内弹出窗口
- **邮件通知**: 可配置邮件提醒

## 高级功能

### 🤖 EA交易（智能交易系统）
- **安装EA**: 将.mq4或.ex4文件复制到Experts目录
- **启用EA**: 工具栏"自动交易"按钮点亮
- **参数设置**: 双击图表上EA名称进行配置
- **回测测试**: 策略测试器进行历史数据回测

### 📜 脚本和指标
- **自定义指标**: 技术分析扩展
- **交易脚本**: 一次性执行的操作
- **安装方法**: 文件放入相应的MQL4目录
- **使用方法**: 从导航窗口拖拽到图表

### 🧪 策略测试器
1. **打开测试器**: 查看 → 策略测试器 或 F6
2. **选择EA**: 从列表中选择要测试的EA
3. **配置参数**: 设置测试时段、时间周期、初始资金
4. **开始测试**: 点击开始按钮，查看测试结果
5. **分析报告**: 查看净利润、胜率、最大回撤等

## 实用技巧

### 快捷键大全
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **F1** | 帮助 | 打开帮助文档 |
| **F2** | 历史数据中心 | 管理历史数据 |
| **F3** | 全局变量 | 查看全局变量 |
| **F4** | MetaEditor | 打开MQL4编辑器 |
| **F6** | 策略测试器 | 打开EA测试工具 |
| **F7** | 编辑EA参数 | 修改当前EA设置 |
| **F8** | 图表属性 | 调整图表显示设置 |
| **F9** | 新订单 | 快速打开下单窗口 |
| **F10** | 市场报价 | 显示/隐藏市场报价 |
| **F11** | 全屏模式 | 切换全屏显示 |
| **F12** | 步进图表 | 向左移动一根K线 |

### 自定义模板
1. **创建模板**: 配置好图表样式后，右键图表 → 模板 → 保存模板
2. **应用模板**: 打开新图表，右键 → 模板 → 选择模板
3. **共享模板**: 模板文件保存在templates目录下

### 数据管理
- **下载历史数据**: 工具 → 选项 → 图表 → 下载历史数据最大K线数
- **导出报价数据**: 文件 → 打开数据文件夹 → history → 对应经纪商
- **清理缓存**: 定期清理不需要的历史数据

## 故障排除

### 常见问题及解决方案

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| **无法连接服务器** | 网络问题、服务器维护 | 检查网络、更换服务器、等待维护结束 |
| **图表数据不全** | 历史数据缺失 | 工具 → 历史数据中心 → 下载 |
| **EA不执行交易** | EA未启用、权限不足 | 启用自动交易、检查EA设置 |
| **平台运行缓慢** | 内存不足、指标过多 | 关闭不必要图表、减少指标数量 |
| **订单被拒绝** | 价格变动、保证金不足 | 检查账户余额、调整订单参数 |

### 连接问题排查
1. **检查网络连接** - ping服务器地址
2. **验证账户状态** - 登录网页版查看账户状态
3. **防火墙设置** - 允许MT4通过防火墙
4. **代理服务器** - 如果使用代理，正确配置代理设置

- **完整交易功能**: 开仓、平仓、修改订单
- **实时图表**: 支持多种时间周期和图表类型
- **技术指标**: 内置主要技术指标

- **离线模式**: 无网络时查看历史数据和图表

## 安全建议

### 账户保护
1. **强密码**: 使用复杂密码，定期更换
2. **双重验证**: 如果支持，启用双重验证
3. **登录历史**: 定期检查登录记录
4. **安全退出**: 不使用时应退出平台

### 软件安全
1. **官方渠道**: 仅从官方网站下载
2. **定期更新**: 及时安装最新版本
3. **防病毒软件**: 保持防病毒软件更新
4. **备份设置**: 定期备份模板、EA和配置文件
