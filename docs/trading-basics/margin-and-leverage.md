# 保证金与杠杆

保证金和杠杆是外汇保证金交易的核心概念，直接决定了您的交易规模和风险敞口。

## 什么是保证金？

保证金（Margin）是您进行交易所需要存入的押金。它不是交易费用，而是您账户中被"锁定"用于维持开仓头寸的资金。

### 保证金类型

| 类型 | 说明 |
|------|------|
| **已用保证金** | 当前所有开仓头寸占用的保证金总额 |
| **可用保证金** | 账户净值中未被占用的部分，可用于开新仓 |
| **保证金比例** | 净值 ÷ 已用保证金 × 100%，反映账户健康度 |

## 什么是杠杆？

杠杆（Leverage）是一种乘数效应，允许您用较小的保证金控制较大的交易头寸。

### 杠杆计算公式

```
实际交易金额 = 保证金 × 杠杆倍数
```

### 杠杆对盈亏的影响

以 EUR/USD 1.0850 买入 1 标准手，价格波动 100 pips 为例：

| 杠杆 | 所需保证金 | 盈亏 (100 pips) | 收益率 |
|------|-----------|----------------|--------|
| 1:1 | $108,500 | $1,000 | 0.92% |
| 1:100 | $1,085 | $1,000 | 92.2% |
| 1:500 | $217 | $1,000 | 460.8% |

:::danger 重要提醒
杠杆放大收益的同时也同等放大亏损。在上例中，如果价格反向波动 100 pips：
- 1:100 杠杆下，亏损占保证金的 92.2%
- 1:500 杠杆下，您将损失超过全部保证金
:::

## 保证金计算示例

交易 EUR/USD，1 标准手，杠杆 1:100：

```
合约价值 = 100,000 × 1.0850 = $108,500
所需保证金 = $108,500 ÷ 100 = $1,085
```

## 如何选择合适的杠杆？

| 交易者类型 | 建议杠杆 | 原因 |
|-----------|----------|------|
| 初学者 | 1:10 - 1:50 | 降低风险暴露 |
| 有经验者 | 1:50 - 1:200 | 平衡收益与风险 |
| 专业交易者 | 1:200 - 1:500 | 精细化风险管理能力 |

:::tip 建议
新手请从低杠杆开始。在充分理解市场风险并获得稳定交易经验后，再逐步调整杠杆比例。
:::
