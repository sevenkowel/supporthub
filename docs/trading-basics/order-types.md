# 订单类型

了解不同的订单类型可以帮助您更有效地管理交易。

## 市价单 (Market Order)

以当前市场最优价格立即执行的订单。

- **优点**：即时成交
- **缺点**：在剧烈波动时可能出现滑点

```
操作：买入 0.1 EUR/USD 市价单
结果：以当前 Ask 价 1.0850 立即成交
```

## 挂单 (Pending Order)

设置特定价格，当市场价格达到时自动执行的订单。

### 限价买入单 (Buy Limit)

在低于当前价格的指定位置挂买入单。

```
当前价格：1.0850
Buy Limit：1.0820（等待价格下跌到 1.0820 时买入）
```

### 限价卖出单 (Sell Limit)

在高于当前价格的指定位置挂卖出单。

```
当前价格：1.0850
Sell Limit：1.0880（等待价格上涨到 1.0880 时卖出）
```

### 买入止损单 (Buy Stop)

在高于当前价格的指定位置挂买入单，用于突破交易策略。

### 卖出止损单 (Sell Stop)

在低于当前价格的指定位置挂卖出单。

## 止损和止盈

| 类型 | 说明 |
|------|------|
| **止损 (Stop Loss)** | 当价格达到设定水平时自动平仓，限制亏损 |
| **止盈 (Take Profit)** | 当价格达到设定水平时自动平仓，锁定利润 |

:::warning 强烈建议
每笔交易都必须设置止损！这是风险管理的最基本要求。
:::

## 追踪止损 (Trailing Stop)

止损价格会随市场价格有利变动而自动调整，但不会反向移动。

```
设置追踪止损 30 pips：
- 价格上涨时，止损价自动跟随上移
- 价格下跌时，止损价保持不变
```

## 订单有效期

| 类型 | 说明 |
|------|------|
| **GTC (Good Till Cancel)** | 直到手动取消前一直有效 |
| **Day Order** | 当日有效，交易日结束时自动取消 |
| **指定时间** | 在指定日期和时间过期 |
