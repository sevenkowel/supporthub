# 隔夜利息（Swap / Rollover）完整指南

在外汇交易中，如果您的持仓过了每日结算时间（通常是 **22:00 GMT**），账户会产生一笔**隔夜利息**（Swap，也叫 Rollover）。它可能是正值（您收钱），也可能是负值（您付钱）。

## 📖 什么是 Swap？

Swap 源于外汇的本质——交易货币对实际上是借入一种货币、贷出另一种货币。两种货币的利率差异决定了 Swap 的方向和金额。

**举例说明**：
- 做多 EUR/USD → 相当于借入 USD（支付美元利率）、持有 EUR（获得欧元利率）
- 如果欧元利率 > 美元利率 → 收取正 Swap
- 如果美元利率 > 欧元利率 → 支付负 Swap

---

## 🧮 Swap 计算公式

### 标准计算方式

```
Swap = (交易量 × 合约规模 × Swap点数) ÷ 10
```

**示例**：
- 品种：EUR/USD
- 做多 Swap：-0.52 点/手/天
- 交易量：1 手（100,000 单位）

```
日 Swap = 1 × (-0.52) ÷ 10 = -$0.052
```

### 三倍 Swap 日（周三）

由于外汇市场周末不结算，但利息继续累积，**每周三的 Swap 通常是正常值的 3 倍**，用于覆盖周五和周六的利息。

| 日期 | Swap 倍数 |
|------|----------|
| 周一 | 1× |
| 周二 | 1× |
| **周三** | **3×** |
| 周四 | 1× |
| 周五 | 1× |

## 📊 如何查看各品种的 Swap 利率

### MT4 / MT5 查看方法
1. 在市场报价窗口右键点击品种
2. 选择 **"品种规格"**（Symbol Specification）
3. 查看 **"做多利息"** 和 **"做空利息"** 字段

### 网页端查看
1. 登录账户后台
2. 进入 "交易条件" 或 "产品规格" 页面
3. 筛选品种查看 Swap 利率

## 🕐 Swap 结算时间

| 平台时间（服务器时间） | 说明 |
|----------------------|------|
| **22:00 GMT** | 标准结算时间（夏令时期间） |
| **00:00 EET** | 平台服务器时区（东欧时间） |

> **重要**：Swap 在**持仓跨越结算时间**时产生。如果您在 21:59 开仓，22:01 平仓，也会产生一天的 Swap。

## 🔢 Swap 对交易策略的影响

### 短线交易（日内）
日内交易不过夜，**不产生 Swap**，无需考虑此因素。

### 中长线持仓
长期持仓的 Swap 累积不容忽视：
- 持仓 30 天，每天 -$5 Swap → 累计 -$150
- 选择正 Swap 方向可以成为额外收益

### 套息交易（Carry Trade）
专门利用 Swap 获利的策略：
- **做多**高利率货币 / **做空**低利率货币
- 每天自动收取正 Swap
- 需注意汇率风险可能超过 Swap 收益

## ❓ 常见问题

### Swap 是每天都扣吗？
是的，每个工作日持仓跨越结算时间都会扣取（或收取）Swap。周三会扣取 3 倍。

### 为什么做多和做空的 Swap 都是负数？
当两种货币的利率差异很小，或经纪商手续费附加较高时，做多和做空都可能为负值。

### Swap 扣在哪里？
Swap 直接计入您的账户余额，在持仓面板的"隔夜利息"列可以查看每笔持仓的累积 Swap。

### 平仓后 Swap 会归还吗？
不会。已产生的 Swap 已经结算，平仓不影响已扣取的 Swap。

### 可以通过在结算前平仓来规避 Swap 吗？
可以，在结算时间前平仓即可避免当天 Swap。但需要权衡交易成本（点差+佣金）是否值得。
