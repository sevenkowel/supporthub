# 交易基础

掌握外汇保证金交易的核心概念是成功交易的第一步。本章节将帮助您理解关键交易术语和机制。

## 什么是外汇交易？

外汇交易（Forex / FX）是指在不同国家货币之间进行兑换交易。外汇市场是全球最大、流动性最强的金融市场，日均交易量超过 **7.5 万亿美元**。

### 外汇保证金交易

外汇保证金交易允许您通过支付一定比例的保证金（通常为 0.2%-1%），来控制远大于保证金的交易头寸。这称为**杠杆交易**。

:::info 举例说明
假设 EUR/USD 报价为 1.0850，您想交易 1 标准手（100,000 单位）：
- **无杠杆**：需要全额 $108,500
- **1:100 杠杆**：仅需保证金 $1,085
- **1:500 杠杆**：仅需保证金 $217
:::

## 常用货币对

| 分类 | 货币对 | 说明 |
|------|--------|------|
| 主要货币对 | EUR/USD, GBP/USD, USD/JPY | 流动性最高，点差最低 |
| 次要货币对 | EUR/GBP, AUD/NZD, EUR/JPY | 流动性中等 |
| 异国货币对 | USD/TRY, USD/ZAR, EUR/TRY | 流动性较低，点差较大 |
