# 账户类型对比

## 账户概览

| 特性 | Standard | Pro | ECN |
|------|----------|-----|-----|
| 最低入金 | $100 | $1,000 | $5,000 |
| 点差 | 从 1.2 pips | 从 0.6 pips | 原始点差 |
| 佣金 | 无 | 无 | $3.5/手/单边 |
| 最大杠杆 | 1:100 | 1:200 | 1:500 |
| 最低手数 | 0.01 | 0.01 | 0.01 |
| 最高手数 | 100 | 100 | 200 |
| 隔夜费 | 有 | 有 | 有 |
| 执行方式 | 即时执行 | 市价执行 | 市价执行 |

## 如何选择？

### Standard — 适合初学者

- 入门门槛低（$100 起步）
- 固定点差，易于计算成本
- 杠杆 1:100，风险相对可控

### Pro — 适合有经验交易者

- 更低的浮动点差
- 更高杠杆（1:200）
- 无佣金

### ECN — 适合专业交易者

- 原始银行间点差
- 最高杠杆 1:500
- 适合剥头皮和 EA 交易
- 深度市场数据

## 账户升级

您可以在客户中心随时申请升级账户类型：

1. 登录客户中心
2. 进入 **"账户管理"**
3. 点击 **"升级账户"**
4. 选择目标账户类型
5. 补足最低入金要求（如有）

:::info 注意
- 账户升级后不可降级
- 升级过程不会影响现有持仓
- 升级后新的交易参数立即生效
:::
