# 密码与安全

## 密码类型

您的账户有以下密码：

| 密码类型 | 用途 | 修改路径 |
|----------|------|----------|
| 客户中心密码 | 登录网站和客户中心 | 客户中心 → 设置 → 修改密码 |
| 交易密码 | 登录 MT4/MT5 交易平台 | 客户中心 → 账户管理 → 修改交易密码 |
| 只读密码 | 允许他人查看（不可交易）账户 | 客户中心 → 账户管理 → 只读密码 |

## 密码安全建议

- ✅ 使用至少 8 位密码，包含大小写字母、数字和特殊字符
- ✅ 不同平台使用不同密码
- ✅ 每 3 个月更换一次密码
- ❌ 不要使用生日、手机号等容易猜测的密码
- ❌ 不要与他人共享密码

## 双因子认证 (2FA)

强烈建议开启 Google Authenticator 双因子认证：

1. 下载 Google Authenticator App
2. 在客户中心 → 安全设置 → 开启 2FA
3. 扫描二维码或手动输入密钥
4. 输入验证码完成绑定

:::tip 安全建议
开启 2FA 后，即使密码泄露，他人也无法登录您的账户。这是保护账户安全最有效的措施之一。
:::
