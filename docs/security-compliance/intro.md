## 🔒 账户安全

### 多层安全防护

#### 1. 登录安全
- **双重验证（2FA）**: 强制启用，支持Google Authenticator
- **强密码策略**: 至少12位，包含大小写字母、数字和符号
- **登录历史监控**: 实时查看所有登录活动
- **异常登录检测**: 自动识别可疑登录行为

#### 2. 交易安全
- **交易密码保护**: 独立的交易密码系统
- **IP地址限制**: 可设置允许登录的IP范围
- **会话超时**: 空闲自动退出保护
- **API密钥管理**: 精细的API权限控制

#### 3. 操作安全
- **关键操作确认**: 重要操作需要二次确认
- **交易限制设置**: 可自定义每日交易限额
- **提现白名单**: 受信任的银行账户列表
- **实时通知**: 所有账户操作的即时通知

### 安全功能配置指南

#### 启用双重验证
1. 登录账户 → 安全设置
2. 点击"启用双重验证"
3. 使用Google Authenticator扫描二维码
4. 输入生成的6位验证码
5. 保存恢复代码（重要！）

#### 设置登录IP限制
1. 进入账户设置 → 安全 → IP白名单
2. 添加当前IP地址（自动检测）
3. 添加常用的固定IP地址
4. 启用"仅允许白名单IP登录"

#### 配置交易提醒
1. 通知设置 → 交易提醒
2. 选择接收方式：邮件、短信、应用推送
3. 设置提醒条件：登录、交易、提现等
4. 测试通知功能

## 💰 资金安全

### 客户资金隔离
- **独立银行账户**: 客户资金存放在隔离的银行账户
- **定期审计**: 每月由独立审计师核查
- **资金保护**: 符合监管要求的资金保护措施
- **透明度**: 可随时查看资金存放证明

### 负余额保护
- **自动止损**: 系统自动平仓防止负余额
- **风险限额**: 基于账户类型设置风险限额
- **追加保证金通知**: 提前预警，避免强制平仓
- **保险覆盖**: 部分账户类型提供负余额保护保险

### 提现安全流程
1. **身份验证**: 每次提现需要身份验证
2. **双重确认**: 邮件或短信确认提现请求
3. **处理时间**: 标准处理时间1-3个工作日
4. **安全审核**: 所有提现经过安全团队审核
5. **银行验证**: 提现到新银行账户需要额外验证

## 📋 风险披露

### 交易风险
外汇和差价合约（CFD）交易存在高风险，可能导致您损失所有投资资金。重要风险包括：

#### 市场风险
- **价格波动**: 市场价格可能快速大幅波动
- **杠杆效应**: 放大收益同时也放大亏损
- **流动性风险**: 市场流动性不足时可能难以平仓
- **隔夜费用**: 持仓过夜可能产生额外费用

#### 技术风险
- **平台故障**: 交易平台可能出现技术问题
- **网络延迟**: 网络问题可能导致交易延迟
- **报价差异**: 不同数据源可能存在报价差异
- **系统维护**: 定期维护期间无法交易

### 风险管理建议
1. **教育先行**: 充分了解产品特性再开始交易
2. **风险资本**: 只使用您可以承受损失的资金
3. **止损设置**: 每笔交易都设置合理的止损
4. **仓位管理**: 控制单笔交易的风险敞口
5. **情绪控制**: 避免情绪化交易决策
