# 流动性（Liquidity）

**流动性**是指资产在不显著影响价格的前提下，被快速买卖的能力。外汇市场是全球流动性最高的金融市场，日均交易额超过 **7 万亿美元**。

## 高流动性 vs 低流动性

| 指标 | 高流动性 | 低流动性 |
|------|---------|---------|
| **点差** | 窄（如 EUR/USD 0.1–1 pip） | 宽（可达 10+ pips） |
| **订单执行** | 快速、滑点小 | 慢、滑点大 |
| **价格连续性** | 平滑走势 | 跳空、波动大 |
| **典型场景** | 欧美交叉时段 | 亚洲深夜、节假日 |

## 影响外汇流动性的因素

- **交易时段**：伦敦-纽约重叠时段（13:00–17:00 GMT）流动性最高
- **货币对类型**：主要货币对（EUR/USD、GBP/USD）流动性高于冷门货币对
- **经济事件**：重大数据发布前后流动性暂时降低
- **市场情绪**：极端恐慌或贪婪时流动性骤降

## 低流动性的风险

- **滑点增加**：实际成交价格偏离预期
- **宽点差**：交易成本上升
- **止损穿透**：价格跳空导致止损无法在预设价位执行

## 实用建议

- 尽量在流动性充足的时段交易
- 重大数据发布期间适当减仓
- 对于小众货币对，使用限价单而非市价单
