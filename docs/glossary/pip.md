# Pip（点）

**Pip**（Percentage in Point）是外汇市场中衡量价格变化的**最小标准单位**，通常是价格小数点后第四位（0.0001）。

## 不同品种的 Pip 定义

| 品种类型 | Pip 位置 | 示例 |
|----------|---------|------|
| **EUR/USD、GBP/USD 等主要货币对** | 小数点后第 4 位 | 1.1050 → 1.1051 = 1 pip |
| **USD/JPY、EUR/JPY 等日元对** | 小数点后第 2 位 | 151.50 → 151.51 = 1 pip |
| **黄金（XAU/USD）** | 小数点后第 2 位 | 2300.00 → 2300.01 = 0.01 点 |

> **Pipette（点子）**：部分平台显示 5 位小数，最后一位叫做 Pipette，等于 0.1 pip。

## Pip 值计算

```
每手 Pip 值（报价货币）= 0.0001 × 100,000 = $10
```

**示例**：
- 交易品种：EUR/USD
- 交易量：1 手（100,000 EUR）
- 每点价值 = $10
- 若价格从 1.1000 上涨至 1.1050 = 50 pips
- 盈利 = 50 × $10 = $500

## 为什么 Pip 很重要？

- **衡量盈亏**：每笔交易的盈亏用 pips 来量化
- **评估点差**：EUR/USD 点差 1.2 pips 代表成本为 $12/手
- **设置止损**：如设置 50 pip 止损，知道最大亏损额
