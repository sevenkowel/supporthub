# 保证金追缴（Margin Call）

**保证金追缴**（Margin Call）是指当您的账户净值跌至一定水平时，经纪商发出的预警通知，要求您追加资金或减少持仓，以维持最低保证金要求。

## 触发条件

```
保证金水平（Margin Level）= 净值 ÷ 已用保证金 × 100%
```

当保证金水平跌至 **100%** 时，通常触发 Margin Call 警告。  
当跌至 **50%**（或经纪商设定的止损水平）时，触发**强制平仓**（Stop Out）。

**示例**：
- 账户余额：$1,000
- 开仓 1 手 EUR/USD，保证金需求：$500
- 若净值跌至 $500 → 保证金水平 100% → Margin Call
- 若净值跌至 $250 → 保证金水平 50% → 强制平仓

## 如何应对

1. **追加入金**：将更多资金存入账户提高净值
2. **减少持仓**：手动平仓部分仓位，释放占用保证金
3. **设置止损**：提前设置止损，在损失扩大前自动平仓

## 如何避免 Margin Call

- 合理使用杠杆，不要满仓操作
- 每笔交易风险不超过账户的 1%–2%
- 持续监控账户的保证金水平
- 保留足够的缓冲余额
